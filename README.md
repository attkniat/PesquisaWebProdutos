# PesquisaWebProdutos
Aplicação WEB que permita a pesquisa de um termo e retorne o resultado da pesquisa de acordo com a origem dos dados

Fonte de Dados : www.giraffas.com.br

### Tecnologias utilizadas
- [x] Asp.net Core
- [x] C#
- [x] SQL Server
- [x] Entity Framework 6
- [x] VS Studio 2019
- [x] MSSQL
- [x] .NET CORE 2.1
